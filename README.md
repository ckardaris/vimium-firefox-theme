# What is this?
This is my personal [Vimium](https://github.com/philc/vimium) theme for Firefox.
I have tried to recreate a look similar to the default style of the address bar of Firefox.

I have made the following changes:
- Change the default 2-line per suggestion style of Vimium to a single line.
- Align output columns using CSS grid.
- Remove the "insert text" character, `↪`, from the output.

# How to use?
1. Open Vimium options.
2. Copy the contents of the [vimium.vss](vimium.css) file into the input box under section **CSS for Vimium UI**.
3. Click on `Save changes`.
